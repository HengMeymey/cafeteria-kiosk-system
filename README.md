# Cafeteria Ordering Kiosk System

## 📌 Overview
This project simulates a Cafeteria Ordering Kiosk System using Java.

It demonstrates:
- Multi-threading (concurrent kiosks)
- Locking with ReentrantLock
- Actor-based system design
- Error handling and system simulation

## 🧱 Features
- Place, Change, Cancel Order
- Query Menu
- Menu Management
- Kiosk Start/Shutdown
- Concurrency control (stock & delivery slots)

## ⚙️ Requirements
- Java JDK 8 or higher
